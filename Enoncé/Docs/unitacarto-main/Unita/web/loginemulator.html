Roles will be defined by administrator.

Real Login

Login will be using Shibboleth.
Once we have the Shibboleth user will look in our database for his specific role.
If has no role will be a default user.
TECH: Roles will be able to be configured by super administrator
TECH: Menu will be able to be configured by super administrator
TECH: Login/Roles will be controlled in each screen

Default Users' Role

SUPER_ADMIN:
DN's user will be able to administrate UnitaCarto: create new universities, roles.
ADMIN:
University administrator's user will be able to administrate all the UnitaCarto Data from his university.
STUDENT:
University student's user will be able to see, navigate and build his own plan using "The Education Wikipedia"-UnitaCarto.
DEFAULT:
Default user will be able to see, navigate by "The Education Wikipedia"-UnitaCarto.
